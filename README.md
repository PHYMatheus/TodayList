# 📖 Sobre o Projeto

O **TODAYLIST** é uma aplicação desenvolvida durante o curso da **Rocketseat** com o objetivo de praticar conceitos de desenvolvimento Back-end utilizando **Java** e **Spring Boot**.

A aplicação consiste em um gerenciador de tarefas (*To-do List*), permitindo que usuários organizem suas atividades de forma simples, rápida e eficiente.

Além da implementação das funcionalidades, o projeto serve como laboratório para consolidar conhecimentos sobre arquitetura de APIs REST, persistência de dados, boas práticas de programação e desenvolvimento profissional.

---

# ✨ Funcionalidades

- ✅ Cadastro de usuários
- ✅ Autenticação
- ✅ Criação de tarefas
- ✅ Atualização de tarefas
- ✅ Exclusão de tarefas
- ✅ Listagem de tarefas
- ✅ Organização das atividades diárias
- ✅ API REST

# ⚙️ Como Executar

### Clone o repositório

```bash
git clone https://github.com/SEU-USUARIO/TODAYLIST.git
```

Entre na pasta

```bash
cd TODAYLIST
```

Execute o projeto

```bash
./mvnw spring-boot:run
```

Ou utilize sua IDE favorita.
